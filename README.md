# Snake

Dependencies:
  SDL2
  SDL_IMG
  
To Compile:
 
Download SDL2 and SDL Image from the following links:
  https://libsdl.org/download-2.0.php
  https://www.libsdl.org/projects/SDL_image/

Tutorial on how to install:
http://lazyfoo.net/tutorials/SDL/01_hello_SDL/index.php
http://lazyfoo.net/tutorials/SDL/06_extension_libraries_and_loading_other_image_formats/index.php

If using Visual Studio and still cannot compile, find the following binary files and copy them to same location as the project's .cpp files:
  SDL2.dll
  libtiff-5.dll
  SDL2_image.dll
  zlib1.dll
  libjpeg-9.dll
  libpng16-16.dll
  libwebp-7.dll
These will be found SDL2-verNo/lib/x86
